# toolfetch-rust
roolfetch GUI in rust

to run:  
```bash
cargo run
```
to compile:  
```bash
cargo build --release
```  
the executable will be in target/release/  

# dependencies
**druid**   (gui toolkit)  
**libmath** (round function)  
**winapi**  (dialog function)  
*platform-specific:*  
**num_cpus** (macos)
